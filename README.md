# snake
webpack^3升级前 改良版贪吃蛇，
  ##### webpack@3.11.0
  ##### webapck-dev-server@2.11.1
  
#### 使用
1. clone or download
2. 命令行里，cd到sanke文件夹下
3. 首先全局安装：
   ##### npm install webpack@3.11.0 -g
   ##### npm install webpack-dev-server@2.11.1 -g
   
4. 再snake下：cnpm install 下载node-modules
5. 开启webpack server：npm run start
##### 浏览器localhost:8080访问。
